# AI伙伴岛

文档对齐状态：2026-07-06 修复版。当前文档已对齐到“简单可玩、家长安全、AI 伙伴沟通练习”这一版。

《AI伙伴岛》是一个给孩子和家长一起使用的本地 AI 启蒙小游戏。

核心目标：

```text
让孩子认识 AI，学习如何和 AI 沟通、交流、做朋友，并在安全边界内和 AI 互相成就、互相学习。
```

这里的“做朋友”不是把 AI 当真人，而是让孩子学会：

- 说出自己的愿望。
- 把细节讲清楚。
- 请 AI 先帮第一版。
- 检查 AI 有没有猜错。
- 反馈哪里要改。
- 最后由自己决定。

## 快速打开

在线试玩版：

```text
https://jingzaoxin-cell.github.io/ai-partner-island/
```

说明：

- 在线试玩版由 GitHub Pages 提供，是纯静态前端版本。
- 基础关卡、愿望练习、电子宠物、成长记录等可以直接玩。
- 真实 Codex/OpenAI 导演草稿不在网页端直接调用，避免把 API key 放进浏览器。
- 完整 AI 版请按下面方式在本地或家庭小主机上启动服务端。

主游戏文件：

```text
AI伙伴岛.html
```

本地服务目录：

```text
ai-partner-island-server
```

启动服务：

```bash
cd ai-partner-island-server
npm install
npm start
```

打开游戏：

```text
http://localhost:8787/AI伙伴岛.html
```

如果当前服务跑在 8788，也可以打开：

```text
http://localhost:8788/AI伙伴岛.html
```

如果是从 GitHub 克隆：

```bash
git clone https://github.com/jingzaoxin-cell/ai-partner-island.git
cd ai-partner-island/ai-partner-island-server
npm install
npm start
```

然后打开：

```text
http://localhost:8787/AI伙伴岛.html
```

## 当前版本重点

- 孩子第一眼先玩，不先读长说明。
- 首页保留“今日冒险”和“伙伴回信”。
- 右下角新增“提示”悬浮指引：现在做、点哪里、完成标志；“带我点下一步”只滚到目标，不替孩子通关。
- 愿望卡有“今日小惊喜”，点击后会出现贴纸奖励、宠物心情和下一步小动作；愿望引擎支持故事、画面、游戏、发明、为什么、侦探六类方向。
- 复杂玩法收进“打开更多玩法”和“成长工具箱”。
- “说一句愿望”会自动展开 AI 伙伴对话练习，“孵化我的宠物”不会跳过前 3 个 AI 基础任务。
- 更多玩法和成长工具箱会记住展开状态，孩子连续操作时不会反复收起。
- 家长“更多设置”已和孩子界面分离，不会再隐藏整个页面。
- 手机宽度下孩子任务贴纸板为单列，避免底部窄卡和文字挤压。
- AI 草稿先进家长预览。
- 浏览器不保存 API key。
- 孩子始终是作品的小队长，AI 只是帮第一版。

## 检查方式

服务端测试：

```bash
cd ai-partner-island-server
npm test
```

视觉改动后重点检查：

- 右侧不能大片空白。
- 中间和左侧不能过挤。
- 底部卡片要平铺。
- 中等窄屏按钮不能变成竖排。
- 移动端不能重叠。
- 家长“更多设置”不能隐藏页面。
- 更多玩法和成长工具箱操作后不能自动收起。
- “发明 / 侦探”愿望卡不能显示故事文案。
