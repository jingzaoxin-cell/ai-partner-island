# AI伙伴岛电子宠物向导说明

文档对齐状态：2026-07-06 修复版。本文是家长和孩子看的简化玩法说明；技术维护以 `README.md`、阶段收尾文档和维护文档为准。

这份说明只讲家里怎么玩。目标不是快速通关，而是让孩子在电子宠物的陪伴下认识 AI、学习 AI、应用 AI，并学会和 AI 沟通配合。

更短的玩家入口见：

```text
AI伙伴岛-快速上手与通关秘籍.md
```

在线试玩版：

```text
https://jingzaoxin-cell.github.io/ai-partner-island/
```

在线试玩版可以直接玩基础关卡、愿望练习、电子宠物和成长记录；真实 Codex/OpenAI 草稿需要本地完整 AI 版，不能把 API key 放进浏览器。

## 最简单玩法

每次打开游戏，只做 3 步：

```text
跟宠物认识 AI
-> 练习和 AI 交流
-> 一起应用 AI
```

孩子只要说一句：

```text
我想让 AI 帮我……
```

就可以开始。

## AI 能帮孩子什么

- 想一个故事。
- 设计一个电子宠物。
- 把想象变成图像提示词。
- 设计一个小游戏规则。
- 用生活例子解释知识。
- 出几个小练习。
- 帮孩子检查一个回答是不是真的。

## 怎么在游戏里学知识

不要先问孩子“你要学数学吗”。先让孩子做东西。

比如做电子宠物时，孩子设置这些参数：

- 多大：学习大小、长度、比例和单位。
- 什么颜色：练习把颜色说具体。
- 什么形状：认识圆形、三角形、对称。
- 怎么移动：理解跳、飞、滑、滚和速度。
- 吃什么能量：认识能量、材料和变化。
- 英文名字：练习简单英语。

孩子感觉是在创造宠物，其实在练数学、语文表达、英语、科学和物理，还在练习怎么把要求告诉 AI。

## 孩子要负责什么

- 说出目标。
- 加上细节。
- 选择喜欢的草稿。
- 改成自己的说法。
- 检查 AI 有没有乱猜。
- 不告诉 AI 学校、住址、电话和真实照片。

## 语音怎么玩

支持语音的浏览器里，可以点“点我说话”。

孩子可以说：

- “读故事”
- “下一步”
- “我想让 AI 帮我画一只电子宠物”
- “请 Codex 帮忙”
- “完成任务”
- “打开作品展”

如果孩子说的不是口令，游戏会把它当作成长日记记下来。

## Codex 怎么用

在线试玩版里，“请求 Codex 导演”不会连接真实 AI。完整 AI 版需要先启动本地服务：

```bash
cd ai-partner-island-server
npm install
npm start
```

然后按下面流程玩：

1. 家长先点“检测服务”。
2. 再点“请求 Codex 导演”。
3. Codex 会生成草稿。
4. 草稿先进家长预览。
5. 家长觉得安全合适，再采用到成长日记。
6. 孩子继续修改，保留自己的决定。

## 什么时候算完成一次成长

不看速度，看孩子能不能说清这 3 句：

- 我想让 AI 帮我做的是……
- AI 帮了我这一步……
- 我自己决定的是……

## 10 个成长主题

1. 唤醒 AI 小精灵：AI 是助手，不是魔法。
2. AI 不是魔法：AI 会猜，也可能猜错。
3. 提示词魔法卡：说清楚，AI 才更懂。
4. 电子宠物诞生：设计自己的电子宠物。
5. 故事导演工作室：AI 写草稿，孩子改剧情。
6. 想象变成画：用语言描述画面。
7. 设计我的小游戏：角色、目标、规则、奖励。
8. 创建学习伴侣：AI 解释知识，不替写作业。
9. AI 真假侦探：先想一想，再查一查。
10. 家庭 AI 作品展：展示 AI 帮了什么，我决定了什么。

## 家长只需要记住

- 少讲道理，多让孩子提出愿望。
- 不追求完整作品，先保护兴趣。
- 每次只玩 15 到 25 分钟。
- AI 草稿不是标准答案，孩子的修改更重要。
